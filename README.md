# ruban-flow
鲁班工作流引擎，基于springboot的分布式高效工作流引擎
## 流程引擎【80%】
完成流程流转【提交、拿回、打回、退回、挂起、唤醒、催办、转办、协办、否决、跳转、委托、抄送、会签等等】  
节点路由支持【普通节点、单选节点、多选节点、条件单选节点、条件多选节点、自动运行节点、汇总节点、开始节点、结束节点】
